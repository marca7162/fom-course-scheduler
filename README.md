# fom-course-scheduler
Course scheduling and classroom allocation system built with Python, Pandas, and SQLite for Fundación Ortega-Marañón International Programs.

The project uses Python, Pandas, and SQLite to process enrollment data, instructor availability, classroom resources, and course information. The goal is to automate course scheduling while minimizing conflicts and efficiently allocating classrooms and teaching resources.

## Features

- Student enrollment management
- Instructor availability tracking
- Course and credit management
- Classroom allocation
- SQLite database integration
- Data cleaning and normalization
- Scheduling conflict detection
- Resource optimization
